CARA MEMBUAT PHPMYADMIN DAPAT DI REMOTE / MEMBUKA REMOTE DATABASE
================================
1. PASTIKAN DALAM 1 JARINGAN ATAU IP PUBLIK
2. BUKA XAMPP CONTROL PANEL
3. STOP SEMUA PROSES (APACHE,MYSQL DLL)
4. KLIK CONFIG, PADA APACHE (DI XAMPP CONTROL PANEL)
5. PILIH APACHE (HTTPD-XAMPP.CONF)
6. CARI PHPMYADMIN, SEPERTI SYNTAX DIBAWAH:

<Directory "C:/xampp/phpMyAdmin">
        AllowOverride AuthConfig
        Require local
        ErrorDocument 403 /error/XAMPP_FORBIDDEN.html.var
    </Directory>

7. GANTI REQUIRE LOCAL, MENJADI REQUIRE ALL GRANTED
8. SIMPAN DAN JALANKAN KEMBALI APACHE DAN MYSQL
9. PING/BUKA MELALUI WEB BROWSER DENGAN IP DAN PORTNYA/URLNYA
10. SELESAI

* JIKA MASIH BELUM TERKONEKSI, PERIKSA PORTNYA. BUKA PORT MELALUI FIREWALL
